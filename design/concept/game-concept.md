## Elevator Pitch

> 这是一款可爱的猫咪塔防幸存者游戏。你控制一只猫咪英雄在战场上移动，自动攻击不断涌来的捣蛋敌人，同时放置防御塔来辅助防守。在敌人越来越强的波次中存活下来，通过选择升级来构建你的专属build，体验从"勉强存活"到"碾压全场"的成长快感。

**10秒测试**：你是一只可爱的猫咪战士，用可爱的小鱼干和毛线球攻击敌人，放置猫咪塔防设施，在无尽敌潮中存活并不断变强。

## Core Fantasy

**你是一只守护猫粮仓库的英雄猫咪！**

作为最后一道防线，你的爪子就是最可靠的武器。看着自己逐渐变强——从只能发射小鱼干的小猫咪，成长为释放火焰风暴的喵星使者。每一波敌人的进攻都更加凶猛，但你的build也越来越疯狂。这种"我的策略生效了"的满足感，就是游戏的灵魂。

---

## Unique Hook

**幸存者 auto-fire + 塔防辅助的混合体验**

大多数幸存者游戏让你被动挨打然后变强。《喵族幸存者》给你 **主动移动** 和 **放置防御塔** 的控制感。塔不只是辅助——是你的技能延伸，是你策略的一部分。

> "像《又一个僵尸幸存者》一样自动攻击变强，**同时还能放置防御塔**来构建专属防线。"

## Player Experience Analysis (MDA Framework)

### Target Aesthetics（目标美学）

| 美学 | 优先级 | 如何实现 |
| ---- | ---- | ---- |
| **Sensation** (感官愉悦) | ⭐⭐⭐⭐ | 可爱的视觉风格、爽快的攻击特效、金币飞出的满足感 |
| **Fantasy** (幻想) | ⭐⭐⭐ | 扮演超级猫咪战士的代入感 |
| **Narrative** (叙事) | ⭐ | 简单背景故事，主要聚焦游戏体验 |
| **Challenge** (挑战) | ⭐⭐⭐⭐⭐ | 波次递增的难度、build优化的策略深度 |
| **Fellowship** (社交) | ⭐ | 单人游戏，主要通过解锁收集满足 |
| **Discovery** (探索) | ⭐⭐⭐ | 尝试不同猫咪和塔的组合、发现强力build |
| **Expression** (表达) | ⭐⭐⭐⭐ | 构建不同build的创造性、每局不同的体验 |
| **Submission** (放松) | ⭐⭐⭐ | 轻松的可爱画风、自动战斗减少操作压力 |

### Key Dynamics（涌现行为）

- 玩家会主动寻找最优位置（敌人多但不被包围）
- 玩家会实验不同的升级组合来发现强力synergy
- 玩家会在失败后分析"如果早点拿那个升级..."
- 玩家会分享强力build给朋友

### Core Mechanics（核心机制）

1. **移动 + 自动攻击** — 玩家控制猫咪移动，猫咪自动向最近的敌人发起攻击
2. **波次系统** — 敌人按波次涌来，每波更强
3. **升级选择** — 每波结束或积累经验时，从3-4个升级中选择一个
4. **塔位放置** — 地图上有多个塔位，放置防御塔辅助输出或控制
5. **成长系统** — 解锁新猫咪、新塔、永久升级

## Player Motivation Profile

### Primary Psychological Needs（主要心理需求）

| 需求 | 如何满足 | 强度 |
| ---- | ---- | ---- |
| **Autonomy（自主性）** | 每波升级选择、塔位选择、英雄选择 | 核心 |
| **Competence（能力感）** | 从"勉强存活"到"碾压全场"的成长感 | 核心 |
| **Relatedness（关联感）** | 喜欢自己的猫咪英雄、有收集新猫咪的期待 | 支持 |

### Player Type Appeal（玩家类型）

- ✅ **Achievers（成就者）** — 解锁新猫咪、完成挑战、达到更高波次
- ✅ **Explorers（探索者）** — 尝试所有build组合、发现隐藏内容
- ❌ **Socializers（社交者）** — 单人游戏，无社交功能
- ❌ **Killers/Competitors（竞争者）** — 无PVP和排行榜

### Flow State Design（心流设计）

- **新手引导曲线**：前5波教学基本操作，让玩家熟悉移动和塔位
- **难度递增**：波次越高敌人越强，但玩家也在变强
- **反馈清晰**：数值增长、视觉效果、特效升级让玩家感受到成长
- **失败恢复**：死亡后立即重新开始，低挫败感高期待

## Core Loop

### 30秒循环（瞬间体验）

```
玩家移动定位
    ↓
猫咪自动攻击最近敌人
    ↓
敌人死亡 → 金币/经验飞出
    ↓
玩家躲避敌人 + 找最佳位置
    ↓
（循环）
```

**核心动作**：移动定位 —— 这个动作需要策略：找到能打到最多敌人、又不会被包围的位置。

**内在满足感**：攻击的节奏感、走位的紧迫感、数字上涨的快感。

### 5分钟循环（短期目标）

```
进入一波敌人
    ↓
战斗（约30-60秒）
    ↓
波次结束 → 经验结算
    ↓
选择升级（3-4选1）
    ↓
Build变强
    ↓
进入下一波（敌人更强）
    ↓
（循环）
```

**玩家决策**：
- 强化英雄攻击 vs 增加防御塔 vs 生存能力
- 在哪里放置塔

### 会话循环（一局游戏）

```
选择猫咪英雄
    ↓
第1波 → 第2波 → ... → 第10波
    ↓
Boss波（高潮挑战）
    ↓
无尽波（可选）
    ↓
死亡/通关 → 结算展示
    ↓
解锁新猫咪/塔
    ↓
"再来一局"（新build尝试）
```

**自然停止点**：死亡后的结算画面
**回归钩子**："再玩3局就能解锁榴莲猫了"

### Long-Term Progression（长期成长）

- **解锁新猫咪英雄**（每只有独特被动和初始能力）
- **解锁新防御塔类型**
- **解锁永久升级**（meta progression，每次运行都生效的小加成）
- **成就系统**

## Game Pillars

### Pillar 1: 可爱即正义

所有视觉元素——猫咪、敌人、攻击特效、UI——都必须可爱、温暖、让人想截图分享。

*设计测试*：如果一个设计"很酷但不可爱"，放弃它。

### Pillar 2: 成长的爽感

玩家必须频繁感受到变强——每次升级都要有明显的效果反馈。

*设计测试*：如果一个升级"只有5%提升且感觉不到"，重新设计。

### Pillar 3: 策略有深度

build选择和塔位决策必须有意义——不同选择导致明显不同的游戏体验。

*设计测试*：如果"最优解永远一样"，增加选择的权衡成本。

### Anti-Pillars（不做的事）

- **不做复杂操作** — 这是auto-fire游戏，不需要精准操作。深度来自策略选择。
- **不做黑暗/血腥风格** — 可爱支柱禁止恐怖和血腥元素。
- **不做多人/PVP** — 单人开发精力有限，专注于单人体验深度。

## Risks and Open Questions

### Design Risks

- **Build可能太简单或太复杂** — MVP快速测试，根据反馈调整
- **后期内容重复感** — 通过多样化敌人类型和地图主题缓解

### Technical Risks

- **大量敌人同屏性能** — Godot 4的2D性能应足够，预估可支持50+敌人
- **存档系统的meta progression** — Godot有成熟的用户数据持久化方案

### Market Risks

- **幸存者品类竞争激烈** — 猫咪主题 + 塔防元素提供差异化
- **Steam市场曝光** — 需要良好的预告片和可爱截图营销

### Scope Risks

- **内容量可能超出单人开发能力** — 严格按MVP→v1.0→完整愿景阶段递进
- **美术资源生产速度** — 像素风可快速迭代，考虑外包部分资源

### Open Questions

- **猫咪攻击方式**：小鱼干 vs 毛线球 vs 爪击？（需要原型测试手感）
- **塔位数量**：每张地图3个还是5个塔位？（影响策略深度）
- **升级选择数量**：每波3选1还是4选1？（影响build多样性vs决策疲劳）

## MVP Definition

**核心假设**：玩家觉得"移动 + 自动攻击 + 升级选择 + 塔位放置"的循环有趣，愿意玩15分钟以上。

**MVP必须包含**：
1. 1只可玩的猫咪英雄
2. 3种防御塔（攻击型、控制型、辅助型）
3. 5种敌人（小怪3种 + 精英1种 + Boss 1种）
4. 10个升级选项
5. 1张地图（猫粮仓库主题）
6. 10波敌人
7. 基本的UI（血条、经验条、升级选择面板）
8. 简单的结算画面

**MVP明确不做**：
- 多猫咪英雄
- 解锁系统
- 多张地图
- 音效/音乐（可以后加）

### Scope Tiers

| 阶段 | 内容 | 功能 | 时间线 |
| ---- | ---- | ---- | ---- |
| **MVP** | 1英雄+3塔+5敌人+10升级+1地图+10波 | 核心循环验证 | 2-4周 |
| **v1.0** | 6英雄+10塔+15敌人+50升级+3地图+无尽模式 | 正式发布 | 3-6个月 |
| **完整愿景** | 10英雄+12塔+18敌人+60升级+5地图+成就系统 | 长期更新 | 6-12个月 |
